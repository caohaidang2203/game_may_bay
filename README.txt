Game Máy Bay Chiến Đấu sử dụng Visual studio 2012, thư viện SDL, C++
Xem demo tại : https://www.youtube.com/watch?v=6tGa8ITpa18
Học tại: https://phattrienphanmem123az.com/lap-trinh-game-cpp
Các thay đổi: đơn giản hơn, game có menu, điểm, tính thời gian, thay đổi các hình nền
Cách chơi : sử dụng các phím mũi tên lên, xuống, trái, phải để di chuyển nhân vật chính tránh khỏi các đạn địch và máy bay địch.
Nhấn chuột trái để bắn đạn hạ máy bay địch. Mỗi lần hạ máy bay địch được 1 điểm.
